# Simple Calculator

## Goals:

- Learn React
- Learn Spring Boot

## Backend:

- Spring Boot
- Lombok
- Maven
- Hibernate
- MYSQL

## Frontend:

- React
- Bootstrap (CSS - Libary)
- Jest (JS Unit-Testing)
- Node
- NPM

## Communication between frontend and backend:

- HTTP method: GET
- HTTP requests example:
  - http://{ip}/{**number**}/{**operation**}/{**number**}
  - http://{ip}/{**float**}/{**+**|**-**|**:**|**x**}/{**float**}
  - http://{ip}/**-4**/**:**/**14.2**
